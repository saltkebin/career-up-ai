## はじめに

本アプリは、**キャリアアップ助成金（正社員化コース）** の申請業務を支援するためのWebアプリケーションです。

### このアプリでできること

- 賃金3%上昇要件の自動計算
- 支給要件の事前チェック
- 必要書類の過不足確認
- AIによる書類データの自動読み取り
- 申請可否の総合判定
- 申請概要書のPDF出力

### 対象となる助成金コース

**正社員化コース**（有期雇用労働者等を正規雇用労働者に転換または直接雇用）

## 各機能の使い方

### 0. ダッシュボード

**パス:** `/dashboard`（ログイン後に自動遷移）

#### 概要
顧問先企業と申請案件を一元管理する画面です。企業を選択すると、その企業の対象労働者一覧が表示されます。

#### 主な機能

**顧問先企業管理**
- 企業一覧の表示（申請件数、期限間近の件数）
- 新規顧問先の登録
- 企業情報の編集・削除
- キャリアアップ計画届出日、就業規則整備状況の管理

**申請管理**
- 対象労働者の一覧表示
- 新規申請の登録
- 申請詳細の確認・編集
- ステータス変更（準備中→書類作成中→申請済み→承認済み/不承認）
- 申請の削除

**統計情報**
- 顧問先企業数
- 全申請件数
- 期限間近の件数（14日以内）
- 重点支援対象者数
- 想定助成金総額

#### 期限アラート

申請期限が14日以内の案件がある場合、ダッシュボード上部に赤色のアラートが表示されます。

### 1. 賃金上昇率計算

**パス:** `/calculator`

#### 概要
転換前6ヶ月と転換後6ヶ月の賃金データを入力し、3%上昇要件を満たしているかを自動計算します。

#### 操作手順

1. **転換日を入力**
   - 正社員に転換した日付を選択します
   - 「入力フォームを生成」をクリック

2. **転換前6ヶ月の賃金を入力**
   - 各月について以下を入力：
     - **基本給**: 月額基本給
     - **固定的諸手当**: 役職手当など（3%計算対象）
     - **残業代（除外）**: 時間外手当（3%計算から除外）
     - **通勤手当（除外）**: 実費補填的手当（3%計算から除外）
     - **実労働日数 / 所定労働日数**: 日割計算用

3. **転換後6ヶ月の賃金を入力**
   - 同様に入力

4. **「賃金上昇率を計算」をクリック**
   - 結果が表示されます

#### 計算対象について

| 対象となる賃金 | 除外される賃金 |
|--------------|--------------|
| 基本給 | 残業代（時間外手当） |
| 役職手当 | 賞与（臨時的支給） |
| 職務手当 | 通勤手当 |
| その他固定的諸手当 | 住宅手当・家族手当 |

#### デモデータ

初めての方は「デモデータを試す」ボタンで動作を確認できます：
- **3%要件クリア**: 基本給20万→21.7万（約4.8%上昇）
- **3%ギリギリ**: わずかに要件達成
- **3%要件未達**: 要件を満たさないケース
- **手当で3%達成**: 手当増額で要件達成

### 2. 支給要件チェック

**パス:** `/eligibility`

#### 概要
キャリアアップ助成金の支給要件を項目ごとにチェックし、受給資格の有無を事前確認します。

#### チェック項目

**基本要件**
- [ ] キャリアアップ計画の届出（転換前）
- [ ] 就業規則の整備
- [ ] 社会保険の適用
- [ ] 雇用保険の加入
- [ ] 6ヶ月以上の雇用実績

**労働者要件**
- [ ] 転換前の雇用期間（3年以内）
- [ ] 事業主との関係（親族でない等）
- [ ] 過去の転換歴
- [ ] 解雇歴の確認

**賃金要件**
- [ ] 3%以上の賃金上昇
- [ ] 賃金支払日の適正性

#### 重点支援対象者の判定

2025年度から追加された重点支援対象者の該当判定も自動で行います：

- **カテゴリA**: 雇入れ3年以内かつ雇用保険未加入者からの転換
- **カテゴリB**: 派遣労働者からの転換
- **カテゴリC**: 正社員求人に応募したが有期雇用で採用された者

### 3. 書類チェック

**パス:** `/documents` → `/documents/check`

#### 概要
申請に必要な書類が揃っているかをチェックリスト形式で確認します。

#### チェック対象書類（基本）

| No | 書類名 | 備考 |
|----|--------|------|
| 1 | キャリアアップ計画書 | 転換前に届出済みであること |
| 2 | 様式第3号（支給申請書） | 正式な申請書 |
| 3 | 様式第3号・別添様式1-1 | 対象労働者の詳細 |
| 4 | 様式第3号・別添様式1-2 | 賃金情報 |
| 5 | 支給要件確認申立書 | 要件充足の申立 |
| 6 | 就業規則（転換前・後） | 正社員転換規定を含むこと |
| 7 | 雇用契約書（転換前・後） | 両方必要 |
| 8 | 賃金台帳（12ヶ月分） | 転換前6ヶ月＋転換後6ヶ月 |
| 9 | 出勤簿/タイムカード（12ヶ月分） | 同上 |
| 10 | 賃金3%増額計算書 | 上昇率の計算根拠 |
| 11 | 登記事項証明書 | 法人の場合 |

#### 重点支援対象者の場合の追加書類

| No | 書類名 | 対象カテゴリ |
|----|--------|-------------|
| 12 | 重点支援対象者確認票 | A/B/C共通 |

#### デモシナリオ

- **全書類完備**: すべての書類が揃っている状態
- **書類不足（3件）**: 賃金台帳、出勤簿、登記事項証明書が不足
- **記入漏れあり**: 様式第3号の必須項目が未記入

### 4. AI書類分析

**パス:** `/documents/analyze`

#### 概要
賃金台帳や申請書類の画像をアップロードすると、AIが自動でデータを読み取り、抽出します。

#### 対応書類

- 賃金台帳
- 給与明細
- 雇用契約書
- その他申請関連書類

#### 使い方

1. 「ファイルを選択」をクリック
2. 書類の画像（JPEG/PNG）またはPDFをアップロード
3. 「分析開始」をクリック
4. AIが自動でデータを抽出
5. 抽出結果を確認・編集
6. 「賃金計算に反映」で他機能と連携

#### 注意事項

- 画像は鮮明なものをご使用ください
- 手書き文字の認識精度は低下する場合があります
- 抽出結果は必ず目視で確認してください

### 5. 申請可否判定

**パス:** `/summary`

#### 概要
入力されたすべての情報を総合的に判定し、申請可否と予想助成金額を算出します。

#### 判定結果

| ステータス | 意味 |
|-----------|------|
| ✅ 申請可能 | すべての要件を満たしています |
| ⚠️ 要確認 | 一部要件の確認が必要です |
| ❌ 申請不可 | 重大な要件を満たしていません |

#### 助成金額の目安（2025年度）

**基本額（中小企業）**
| 転換パターン | 金額 |
|-------------|------|
| 有期 → 正規 | 80万円 |
| 無期 → 正規 | 40万円 |

**重点支援対象者加算**
| カテゴリ | 加算額 |
|---------|--------|
| カテゴリA | 12万円 |
| カテゴリB | 12万円 |
| カテゴリC | 9.5万円 |

#### デモシナリオ

- **申請可能**: 全要件クリア・重点支援対象者（92万円）
- **賃金要件NG**: 3%上昇要件を満たさない
- **書類不備**: 必要書類が不足

### 6. PDFエクスポート

**パス:** `/export`

#### 概要
入力したデータから申請概要書をPDF形式で自動生成します。

#### 出力内容

1. **事業所情報**
   - 事業所名、住所、電話番号
   - 雇用保険適用事業所番号
   - 代表者名、キャリアアップ管理者名

2. **対象労働者情報**
   - 氏名、生年月日、性別
   - 雇入れ日、転換日
   - 転換前後の雇用形態

3. **賃金情報**
   - 転換前後の賃金額
   - 賃金上昇率

4. **申請金額**
   - 基本額、加算額、合計額

#### 使い方

1. 各項目を入力（デモデータ読込も可能）
2. 「PDFを生成・ダウンロード」をクリック
3. PDFファイルが自動でダウンロードされます

### 7. よくある失敗例ガイド

**パス:** `/guide`

#### 概要
キャリアアップ助成金申請でよくある失敗パターンと対策を解説します。

#### 主な失敗例

1. **残業代を3%計算に含めてしまう**
   - ❌ 残業代は変動給のため計算対象外
   - ✅ 基本給と固定的諸手当のみで計算

2. **キャリアアップ計画を転換後に届出**
   - ❌ 転換日以降の届出は要件を満たさない
   - ✅ 転換日の前日までに届出が必要

3. **就業規則に正社員転換規定がない**
   - ❌ 就業規則への明記がないと要件不備
   - ✅ 転換制度を就業規則に規定

4. **賃金支払日が変則的**
   - ❌ 6ヶ月の比較期間にズレが生じる
   - ✅ 賃金締め日・支払日を正確に把握

5. **書類の保存期間不足**
   - ❌ 必要書類が揃わない
   - ✅ 申請前に12ヶ月分の書類を確保

## 推奨ワークフロー

効率的な申請業務のための推奨手順です。

```
1. 事前確認フェーズ
   └─ 支給要件チェック → 受給資格の確認

2. データ収集フェーズ
   ├─ AI書類分析 → 賃金台帳からデータ抽出
   └─ 書類チェック → 必要書類の確認

3. 計算・検証フェーズ
   └─ 賃金上昇率計算 → 3%要件の確認

4. 総合判定フェーズ
   └─ 申請可否判定 → 最終確認

5. 書類作成フェーズ
   └─ PDFエクスポート → 申請概要書の生成
```

## 2025年度の重点支援対象者について

2025年度から新設された「重点支援対象者」に該当する場合、助成金が加算されます。

### カテゴリA
**雇入れ3年以内かつ雇用保険未加入者からの転換**

- 入社から3年以内であること
- 転換前に雇用保険に加入していなかったこと
- 加算額: 12万円（中小企業）

### カテゴリB
**派遣労働者からの転換**

- 派遣先事業所に直接雇用されること
- 派遣期間が6ヶ月以上であること
- 加算額: 12万円（中小企業）

### カテゴリC
**正社員求人に応募したが有期雇用で採用された者**

- 正社員として応募した記録があること
- 結果として有期雇用で採用されたこと
- 加算額: 9.5万円（中小企業）

## よくある質問（FAQ）

### Q1. ログインしなくても使えますか？

**A.** はい、賃金計算など一部機能はログインなしでもお試しいただけます。ただし、ダッシュボードでの顧問先・申請管理にはログインが必要です。

### Q2. 入力したデータはどこに保存されますか？

**A.** データはブラウザのローカルストレージに保存されます。同じブラウザ・同じ端末であれば、ページを閉じてもデータは保持されます。ただし、ブラウザのキャッシュをクリアするとデータが消去されますのでご注意ください。

### Q3. AI書類分析の精度はどの程度ですか？

**A.** 一般的な賃金台帳で90%以上の認識精度を目指していますが、手書き文字やフォーマットによっては精度が低下する場合があります。必ず結果を目視でご確認ください。

### Q4. 大企業の場合も使えますか？

**A.** はい、大企業の場合は「中小企業以外」を選択してください。助成金額が異なります。

### Q5. 複数の対象労働者を管理できますか？

**A.** はい、ダッシュボード機能で複数の顧問先企業と申請案件を管理できます。企業ごとに労働者を整理して表示できます。

### Q6. 申請期限のリマインダーはありますか？

**A.** ダッシュボードで申請期限を管理し、期限が14日以内の案件はアラートとして表示されます。

### Q7. パスワードを変更できますか？

**A.** 環境変数 `NEXT_PUBLIC_APP_PASSWORD` で変更可能です。運用環境では独自のパスワードを設定してください。
