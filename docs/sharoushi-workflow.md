# 社労士事務所の業務フロー

キャリアアップ助成金（正社員化コース）における社労士事務所の仕事を解説します。

---

## 全体像：5つのフェーズ

```
┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐
│ Phase1  │ → │ Phase2  │ → │ Phase3  │ → │ Phase4  │ → │ Phase5  │
│ 相談    │    │ 準備    │    │ 転換    │    │ 申請    │    │ 支給後  │
│ 診断    │    │ 支援    │    │ サポート │    │ 代行    │    │ フォロー │
└─────────┘    └─────────┘    └─────────┘    └─────────┘    └─────────┘
```

---

## Phase 1: 相談・診断（営業〜契約）

**「そもそも助成金もらえるの？」を判断する段階**

| やること | 具体的な作業 |
|---------|-------------|
| ヒアリング | 企業の雇用状況、正社員にしたい人がいるか確認 |
| 要件チェック | 「この会社・この従業員は対象になるか？」を判定 |
| 見積もり | 「◯人転換すれば最大◯◯万円もらえます」と提示 |
| スケジュール提示 | 「◯月に転換すれば◯月に申請できます」と説明 |

**社労士の価値:** 複雑な要件を理解し、企業に「できる/できない」を正確に伝える

---

## Phase 2: 事前準備の支援（転換前にやること）

**「申請できる状態」を整える段階**

| やること | なぜ必要？ |
|---------|-----------|
| キャリアアップ計画書の作成・届出 | これを出さないと助成金の対象にならない |
| 就業規則の確認・修正 | 「正社員への転換制度」が書いてないとNG |
| 就業規則の届出 | 労働基準監督署に届け出る（10人以上の会社は必須） |
| 雇用契約書の整備 | 転換前後で「何が変わったか」を証明する書類 |

**ポイント:** これらは全て**転換日の前日まで**に完了が必要！

---

## Phase 3: 転換時のサポート

**「正社員になる日」のアドバイス**

| やること | 具体的な内容 |
|---------|-------------|
| 転換日のアドバイス | 給与締め日を考慮した最適な日を提案 |
| 賃金設計 | 「3%以上上げないとNG」なので計算して確認 |
| 書類作成支援 | 新しい雇用契約書、辞令などの作成 |

**よくある失敗:** 3%上げたつもりが計算方法を間違えて不支給 → 社労士がチェック

---

## Phase 4: 支給申請（メインの仕事）

**「助成金をください」と申請する段階**

### 申請タイミング

```
転換日
  │
  ├── 6ヶ月間、正社員として雇用 ──→ 第1期申請（期限: 賃金支払日から2ヶ月以内）
  │
  └── さらに6ヶ月（計12ヶ月）──→ 第2期申請（重点支援対象者のみ）
```

### 社労士の作業

| やること | 詳細 |
|---------|------|
| 書類収集 | 企業から賃金台帳、出勤簿、契約書などを集める |
| 申請書作成 | 10種類以上の様式に記入（ここが一番手間） |
| 不備チェック | 書類間の整合性、計算ミスがないか確認 |
| 労働局へ提出 | 代理で申請書を提出 |
| 審査対応 | 労働局からの質問・追加資料要求に対応 |

**超重要:** 申請期限（2ヶ月）を1日でも過ぎたら**絶対に受理されない**

---

## Phase 5: 支給決定後のフォロー

**「お金が振り込まれた後」の仕事**

| やること | 詳細 |
|---------|------|
| 支給決定の確認 | 金額・振込日を企業に連絡 |
| 第2期の準備 | 重点支援対象者なら6ヶ月後にもう一度申請 |
| 次の対象者確認 | 「他にも正社員にできる人いませんか？」と提案 |

## 社労士の仕事で「大変なところ」

| 大変なポイント | 理由 |
|---------------|------|
| **書類が多い** | 10種類以上の様式、添付書類も大量 |
| **計算が細かい** | 残業代は1分単位、3%増額の計算も複雑 |
| **期限が厳しい** | 2ヶ月を1日でも過ぎたらアウト |
| **ミスが許されない** | 不正受給は5年間助成金停止+刑事罰の可能性 |
| **制度が毎年変わる** | 2025年も大幅改正、常に最新情報が必要 |

## アプリで効率化できるポイント

| 社労士の悩み | アプリでの解決策 |
|-------------|-----------------|
| 3%計算が面倒 | → 自動計算ツール |
| 期限管理が大変 | → リマインダー通知 |
| 対象者の判定が複雑 | → チェックリスト自動判定 |
| 書類の不備が怖い | → AIによる事前チェック |
| 書類作成に時間がかかる | → テンプレート自動生成 |
