## 1. 制度概要

### キャリアアップ助成金とは
厚生労働省が提供する助成金制度で、非正規雇用労働者（有期雇用労働者・短時間労働者・派遣労働者）の企業内でのキャリアアップを促進することを目的としています。

### 正社員化コースとは
有期雇用労働者等を正規雇用労働者（正社員）に転換または直接雇用した事業主に対して助成金を支給する制度です。

**ポイント:**
- 従業員1人当たり最大80万円（中小企業の場合）
- 年間20名まで申請可能
- 2期（12ヶ月）に分けて支給

---

## 2. 2025年度の主な変更点

### 2.1 「重点支援対象者」区分の新設
2025年4月から、対象労働者に「重点支援対象者」という新しい区分が設けられました。

**重点支援対象者の定義:**

| 区分 | 条件 |
|------|------|
| A | 雇い入れから**3年以上**の有期雇用労働者 |
| B | 雇い入れから3年未満で、以下の両方を満たす有期雇用労働者<br>①過去5年間に正社員であった期間が合計1年以下<br>②過去1年間に正社員として雇用されていない |
| C | 派遣労働者、母子家庭の母等、父子家庭の父、人材開発支援助成金の特定訓練修了者 |

### 2.2 支給額の変更
- **重点支援対象者以外**の支給額が**半額**に引き下げ
- 重点支援対象者以外は**1期のみ**の申請に制限

### 2.3 キャリアアップ計画書の簡素化
- 従来: 労働局長の「認定」が必要
- 2025年度以降: **「届出」のみ**で可（認定不要）

### 2.4 その他の変更
- 雇用期間が通算5年を超える有期雇用労働者は「無期雇用労働者」として扱う
- 新卒の非正規雇用労働者は入社から1年間は助成金の対象外

## 3. 支給額一覧

### 3.1 基本支給額

| 転換パターン | 対象者区分 | 中小企業 | 大企業 |
|-------------|-----------|---------|--------|
| 有期→正規 | 重点支援対象者 | **80万円**（40万円×2期） | 60万円（30万円×2期） |
| 有期→正規 | 重点支援対象者以外 | **40万円**（1期のみ） | 30万円（1期のみ） |
| 無期→正規 | 重点支援対象者 | **40万円**（20万円×2期） | 30万円（15万円×2期） |
| 無期→正規 | 重点支援対象者以外 | **20万円**（1期のみ） | 15万円（1期のみ） |

### 3.2 加算措置

| 加算項目 | 中小企業 | 大企業 |
|---------|---------|--------|
| 正社員転換制度を新たに規定 | +20万円/事業所 | +15万円/事業所 |
| 多様な正社員制度を新たに規定 | +40万円/事業所 | +30万円/事業所 |
| 人材開発支援助成金の訓練修了後に転換 | 加算あり | 加算あり |

**多様な正社員とは:**
- 勤務地限定正社員
- 職務限定正社員
- 短時間正社員

### 3.3 申請上限
- **1年度につき1事業所あたり20名まで**（同一対象者の2回目申請を除く）

## 4. 対象となる事業主の要件

### 4.1 基本要件

| No | 要件 |
|----|------|
| 1 | 雇用保険適用事業所の事業主であること |
| 2 | 雇用保険適用事業所ごとに**キャリアアップ管理者**を配置していること |
| 3 | **キャリアアップ計画書**を作成し、管轄労働局へ届出していること |
| 4 | 対象労働者の労働条件・勤務状況・賃金支払い状況を確認できる書類を整備していること |
| 5 | 計画期間内にキャリアアップの取り組みを実施すること |

### 4.2 就業規則の要件
- 正社員への転換制度を**就業規則または労働協約**に規定していること
- 転換手続き、要件、転換時期が明示されていること
- 労働基準監督署への届出（10人以上の事業所は必須）

### 4.3 対象外となる事業主
- 労働保険料を滞納している事業主
- 過去1年以内に労働関係法令違反がある事業主
- 過去に不正受給を行った事業主（5年間）
- 性風俗関連事業を営む事業主

## 5. 対象となる労働者の要件

### 5.1 対象となる労働者

| 条件 | 詳細 |
|------|------|
| 雇用形態 | 有期雇用労働者、無期雇用労働者、派遣労働者 |
| 雇用期間 | 「正規雇用労働者と異なる雇用区分の就業規則等」の適用を**6か月以上**受けていること |
| 正社員化後 | 正社員として6か月以上継続雇用されること |
| 賃金 | 転換後6か月間の賃金が転換前6か月間より**3％以上増額**していること |
| 正社員の定義 | 「賞与または退職金の制度」と「昇給」がある雇用形態 |

### 5.2 対象外となる労働者

| No | 対象外条件 |
|----|-----------|
| 1 | 正規雇用または無期雇用を条件に雇用した労働者 |
| 2 | 過去3年以内に正規雇用・無期雇用していた労働者 |
| 3 | 事業主・役員の**3親等以内の親族** |
| 4 | **新卒で雇入れから1年未満**の労働者 |
| 5 | 定年まで1年未満の労働者 |
| 6 | 外国人技能実習生 |
| 7 | 正社員化を約束して雇用された労働者 |

## 6. 申請の流れ

### 6.1 全体スケジュール

```
[準備期間]        [転換前6か月]      [転換]      [転換後6か月]    [申請]
    │                  │               │              │            │
    ▼                  ▼               ▼              ▼            ▼
キャリアアップ    対象労働者を      正社員に      賃金支払い    第1期
計画書提出       有期雇用で        転換          (3%以上増)    支給申請
                 雇用継続
    │
    ├── 就業規則整備・届出
    │
    └── キャリアアップ管理者配置
```

### 6.2 詳細ステップ

| Step | 内容 | タイミング |
|------|------|-----------|
| 1 | キャリアアップ計画書の作成・提出 | 転換日の前日まで |
| 2 | 就業規則に転換制度を規定・届出 | 転換日の前日まで |
| 3 | 対象労働者を有期雇用で6か月以上雇用 | 転換前 |
| 4 | 雇用契約書の作成（転換前後） | 随時 |
| 5 | 正社員への転換実施（基本給3%以上増） | 計画期間内 |
| 6 | 転換後6か月間の賃金支払い | 転換後 |
| 7 | **第1期支給申請** | 6か月分の賃金支払日翌日から**2か月以内** |
| 8 | 転換後7〜12か月目の賃金支払い | 転換後（重点支援対象者のみ） |
| 9 | **第2期支給申請** | 12か月分の賃金支払日翌日から**2か月以内** |

### 6.3 重要な期限

| 項目 | 期限 |
|------|------|
| キャリアアップ計画書の提出 | 正社員転換日の**前日まで** |
| 第1期支給申請 | 転換後6か月分の賃金支払日の翌日から**2か月以内** |
| 第2期支給申請 | 転換後12か月分の賃金支払日の翌日から**2か月以内** |

**注意:** 申請期限を過ぎると一切受理されません。

## 8. 社労士事務所の役割と業務フロー

### 8.1 社労士の役割

**重要:** 助成金の申請代行は**社会保険労務士の独占業務**です。報酬を得て助成金の書類作成や申請代理を行えるのは社労士のみです。

### 8.2 社労士事務所の業務フロー

#### フェーズ1: 初期相談・診断

| タスク | 詳細 |
|--------|------|
| 受給可能性診断 | 企業の雇用状況、対象労働者の有無を確認 |
| 要件チェック | 事業主要件・労働者要件の確認 |
| 概算支給額の算出 | 対象者数と支給額の見積もり |
| スケジュール策定 | 申請までのタイムライン作成 |

#### フェーズ2: 事前準備支援

| タスク | 詳細 |
|--------|------|
| キャリアアップ管理者の選任支援 | 適任者の選定アドバイス |
| キャリアアップ計画書の作成 | 計画内容の策定・様式記入 |
| 就業規則の確認・改定 | 転換制度の規定追加 |
| 労働基準監督署への届出代行 | 就業規則の届出 |
| 雇用契約書の整備支援 | 転換前後の契約書テンプレート作成 |

#### フェーズ3: 転換実施支援

| タスク | 詳細 |
|--------|------|
| 転換日の設定アドバイス | 最適な転換タイミングの提案 |
| 賃金設計支援 | 3%以上増額の計算・確認 |
| 労働条件通知書の作成支援 | 転換後の条件明示 |
| 帳票類の整備指導 | 賃金台帳・出勤簿の記載確認 |

#### フェーズ4: 支給申請

| タスク | 詳細 |
|--------|------|
| 申請書類の作成 | 全ての様式の記入 |
| 添付書類の収集・確認 | 書類の不備チェック |
| 賃金上昇要件の計算・確認 | 3%以上増額の証明 |
| 労働局への申請書提出 | 代理申請 |
| 審査対応 | 追加資料の提出、質問への回答 |

#### フェーズ5: 支給決定後

| タスク | 詳細 |
|--------|------|
| 支給決定通知の確認 | 支給額・振込日の確認 |
| 第2期申請の準備 | 重点支援対象者の場合 |
| 次年度の対象者確認 | 継続的な受給支援 |

### 8.3 社労士報酬の一般的な体系

| 報酬タイプ | 相場 |
|-----------|------|
| 着手金 | 0円〜5万円 |
| 成功報酬 | 助成金額の15%〜25% |
| 顧問契約の場合 | 月額顧問料+成功報酬（割引あり） |

## 9. 申請企業がやるべきこと

### 9.1 事前準備（転換前）

#### 労務管理体制の整備

| タスク | 詳細 | 重要度 |
|--------|------|--------|
| 雇用保険の加入確認 | 全対象労働者が加入していること | 必須 |
| 就業規則の整備 | 転換制度を明記（10人未満でも推奨） | 必須 |
| 賃金台帳の整備 | 正確な記録、過去3年分の保管 | 必須 |
| 出勤簿/タイムカードの整備 | 1分単位での労働時間管理 | 必須 |
| 雇用契約書の整備 | 労働条件の明示 | 必須 |
| 社会保険の適正加入 | 加入要件を満たす労働者の加入 | 必須 |

#### 対象労働者の確認

| チェック項目 | 確認内容 |
|-------------|---------|
| 雇用期間 | 6か月以上の有期雇用実績 |
| 雇用形態 | 正社員と異なる就業規則の適用 |
| 親族関係 | 3親等以内の親族でないこと |
| 過去の雇用 | 過去3年以内に正社員でなかったこと |
| 重点支援対象者該当性 | A/B/Cのいずれかに該当するか |

### 9.2 社労士への提供書類

| No | 書類 | 用途 |
|----|------|------|
| 1 | 会社の登記事項証明書 | 中小企業確認 |
| 2 | 就業規則（現行） | 転換制度の確認 |
| 3 | 賃金規程 | 賃金体系の確認 |
| 4 | 対象労働者の雇用契約書 | 雇用条件の確認 |
| 5 | 対象労働者の賃金台帳（6か月分以上） | 賃金実績の確認 |
| 6 | 対象労働者の出勤簿（6か月分以上） | 勤務実績の確認 |
| 7 | 対象労働者の履歴書 | 過去の正社員経験確認 |
| 8 | 雇用保険被保険者証 | 加入状況の確認 |

### 9.3 日常的にやるべきこと

| タスク | 頻度 | 詳細 |
|--------|------|------|
| 労働時間の正確な記録 | 毎日 | 1分単位での記録 |
| 賃金台帳の作成 | 毎月 | 支給日ごとに正確に記録 |
| 残業代の適正計算 | 毎月 | 1分単位での計算 |
| 雇用契約書の更新 | 契約更新時 | 条件変更があれば作成 |
| 労働条件の書面通知 | 変更時 | 労働条件通知書の交付 |

### 9.4 転換時にやるべきこと

| タスク | 詳細 |
|--------|------|
| 転換日の決定 | 給与締め日との関係を考慮 |
| 賃金の決定 | 転換前より3%以上増額 |
| 新しい雇用契約書の作成 | 正社員としての労働条件明示 |
| 辞令の交付 | 正社員転換の正式通知 |
| 社会保険の変更手続き | 必要に応じて |

## 10. よくある失敗例と注意点

### 10.1 不支給になる主な理由

| No | 失敗例 | 対策 |
|----|--------|------|
| 1 | **残業代の計算ミス**（15分単位など） | 1分単位で計算 |
| 2 | **申請期限の超過** | カレンダー管理を徹底 |
| 3 | **就業規則の届出漏れ** | 労基署への届出確認 |
| 4 | **転換制度の規定不備** | 転換手続き・要件・時期を明記 |
| 5 | **3%増額要件未達** | 計算ツールで事前確認 |
| 6 | **親族を対象者に含める** | 3親等以内は除外 |
| 7 | **過去の正社員を対象に含める** | 3年以内の履歴確認 |
| 8 | **計画書提出前に転換** | 必ず転換前に提出 |
| 9 | **会社都合解雇の発生** | 雇用維持期間中は避ける |
| 10 | **契約社員への賞与支給** | 正社員との待遇差を明確に |

### 10.2 審査のチェックポイント

| 項目 | 審査内容 |
|------|---------|
| 賃金台帳と出勤簿の整合性 | 労働時間と支給額の一致 |
| 残業代の適正支払い | 法定計算との一致 |
| 社会保険の加入状況 | 要件該当者の加入有無 |
| 労働条件の書面交付 | 雇用契約書・労働条件通知書 |
| 就業規則の届出 | 労基署受付印の有無 |

### 10.3 不正受給のペナルティ

| ペナルティ | 内容 |
|-----------|------|
| 全額返還 | 受給した助成金の全額 |
| 延滞金 | 年3%（受給日翌日から返還日まで） |
| 違約金 | 返還額の20% |
| 受給制限 | 5年間、雇用関係助成金の受給不可 |
| 刑事罰 | 詐欺罪、有印私文書偽造罪の可能性 |

## 11. アプリ開発への示唆

### 11.1 アプリで解決できる課題

| 課題 | アプリ機能案 |
|------|------------|
| 対象者の管理が煩雑 | 対象労働者データベース、要件自動チェック |
| 申請期限の管理 | スケジュール管理、リマインダー通知 |
| 書類作成の手間 | 申請書類の自動生成 |
| 賃金計算の確認 | 3%増額計算ツール |
| 書類の不備 | 必要書類チェックリスト |
| 進捗管理 | ステータス管理ダッシュボード |

### 11.2 推奨機能一覧

#### 社労士向け機能
- クライアント企業管理
- 対象労働者データベース
- 申請書類自動生成
- 申請スケジュール管理
- 書類テンプレート管理
- 進捗ステータス管理
- 売上・報酬管理

#### 企業向け機能
- 従業員情報入力フォーム
- 必要書類アップロード
- 申請状況確認
- 通知・リマインダー受信

### 11.3 重要なデータ項目

#### 企業情報
- 会社名、所在地、代表者
- 雇用保険適用事業所番号
- 中小企業/大企業区分
- キャリアアップ管理者情報
- 就業規則の有無・届出状況

#### 対象労働者情報
- 氏名、生年月日
- 雇入れ日
- 雇用形態（有期/無期/派遣）
- 契約期間
- 過去の正社員経験
- 重点支援対象者該当区分
- 転換予定日
- 転換前後の賃金

#### 申請管理情報
- キャリアアップ計画書提出日
- 計画期間
- 転換実施日
- 第1期申請期限・申請日・支給日
- 第2期申請期限・申請日・支給日
- 支給額
