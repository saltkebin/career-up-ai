## Phase 1: 初回相談・ヒアリング

### 📅 2025年1月15日（水）

#### 相談内容

株式会社テックフォワードの高橋社長から、山本社労士事務所に電話相談がありました。

> **高橋社長:**
> 「うちで働いている契約社員の佐藤さんを4月から正社員にしたいんです。キャリアアップ助成金というのがあると聞いたのですが、申請できますか？」

#### ヒアリング項目

山本社労士は以下の情報をヒアリングしました：

| 確認項目 | 回答 |
|---------|------|
| 転換予定日 | 2025年4月1日 |
| 対象者の雇入れ日 | 2024年4月1日（約9ヶ月前） |
| 現在の雇用形態 | 契約社員（1年契約・更新の可能性あり） |
| 転換後の雇用形態 | 正社員（期間の定めなし） |
| キャリアアップ計画の届出 | まだしていない |
| 就業規則の正社員転換規定 | ない（要整備） |
| 転換前の賃金 | 基本給22万円 + 職務手当1万円 |
| 転換後の賃金（予定） | 基本給24万円 + 職務手当1万円 |

#### 社労士の初期判断

- ✅ 転換予定日まで約2.5ヶ月あり、キャリアアップ計画の届出は間に合う
- ✅ 雇用期間が6ヶ月以上3年以下（要件を満たす見込み）
- ⚠️ 就業規則の整備が必要
- ⚠️ 雇用保険加入状況の確認が必要

#### Step 2: 要件チェック

チェックボックスを確認しながら入力：

**基本要件**
- [x] キャリアアップ計画を転換日の前日までに届出 → **未実施（要対応）**
- [x] 就業規則に正社員転換規定あり → **未整備（要対応）**
- [x] 社会保険に加入している → **加入済み**
- [x] 雇用保険に加入している → **加入済み**

**労働者要件**
- [x] 雇用期間6ヶ月以上3年以下 → **12ヶ月（要件OK）**
- [x] 事業主と親族関係でない → **該当なし**
- [x] 過去に同一事業主での正社員歴なし → **該当なし**

**重点支援対象者の確認**
- [ ] 雇入れ3年以内かつ雇用保険未加入だった → **該当しない（加入済み）**
- [ ] 派遣労働者からの転換 → **該当しない**
- [ ] 正社員求人に応募したが有期で採用 → **確認が必要**

```
（正社員への転換）

1. 所属長の推薦があること
2. 面接試験に合格すること
3. 転換時期は原則として毎年4月1日とする
```

**届出日:** 2025年2月14日（転換日の45日前）

## Phase 4: 正社員転換の実施

2. **社会保険の届出**
   - 資格取得届の提出（報酬月額変更）

3. **辞令の交付**
   - 佐藤美咲 殿「正社員に任命する」

| 項目 | 転換前 | 転換後 | 増減 |

## アプリ活用のポイント

### 1. 事前確認で手戻りを防ぐ
- **支給要件チェック** で受給資格を早期に確認
- 不備があれば転換前に対応可能

### 2. 計算ミスを防ぐ
- **賃金上昇率計算** で3%要件を正確に判定
- 残業代・通勤手当の除外を自動処理

### 3. 書類の漏れを防ぐ
- **書類チェック** で必要書類を一覧確認
- 重点支援対象者の追加書類も自動判定

### 4. データ入力の手間を削減
- **AI書類分析** で賃金台帳からデータ自動抽出
- 手入力のミスを軽減

### 5. 申請前の最終確認
- **申請可否判定** で総合的に判定
- 見落としがないか最終チェック

### 6. 書類作成の効率化
- **PDFエクスポート** で申請概要書を自動生成
- クライアントへの報告資料としても活用可能
