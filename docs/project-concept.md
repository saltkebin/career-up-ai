# プロジェクトコンセプト

## 概要

**プロジェクト名:** キャリアアップ助成金（正社員化コース）申請支援アプリ

**コンセプト:**
社労士事務所の事務負担を生成AIで最小化し、キャリアアップ助成金（正社員化コース）の申請をスムーズに完了させるWebアプリケーション

---

## ターゲットユーザー

### プライマリ：社労士事務所
- キャリアアップ助成金の申請代行を行う社会保険労務士
- 複数のクライアント企業を抱えている事務所
- 事務作業の効率化・省力化を求めている

### セカンダリ：申請企業
- 社労士事務所のクライアント企業
- 助成金申請に必要な情報・書類を提供する側

---

## 解決したい課題

### 社労士事務所の課題
1. **説明資料作成の手間** - 企業ごとに制度説明資料を作成
2. **書類チェックの時間** - 提出書類の不備・漏れの目視確認
3. **手入力・転記作業** - 申請書類への情報転記

---

## 主要機能（案）

### 1. 企業向け図解生成
| 項目 | 内容 |
|------|------|
| 概要 | 制度説明・申請フローを図解化したビジュアル資料を自動生成 |
| AIの役割 | テキスト情報から図解画像を生成 |
| 削減効果 | 説明資料作成の手間を削減 |

**想定される利用シーン（要検討）:**
- 新規顧客への制度説明時（営業ツール）
- 契約後の企業担当者への手順説明
- 企業が社内で従業員に説明する用

### 2. 提出書類チェック
| 項目 | 内容 |
|------|------|
| 概要 | 提出書類の不備・漏れ・整合性をAIがチェック |
| AIの役割 | アップロードされた書類を解析し、問題点を指摘 |
| 削減効果 | 目視チェックの時間を削減、審査落ちリスクを低減 |

**チェックレベル（要検討）:**
| レベル | 内容 |
|--------|------|
| Level 1 | 必須項目の記入漏れチェック |
| Level 2 | + 添付書類の過不足チェック |
| Level 3 | + 書類間の整合性チェック（賃金台帳と申請書の金額一致等） |
| Level 4 | + 要件充足チェック（3%増額、6か月雇用等） |

### 3. フォーマットに合わせた書類作成
| 項目 | 内容 |
|------|------|
| 概要 | 入力データから申請書類を自動生成 |
| AIの役割 | データ抽出・フォーマット変換・文章生成 |
| 削減効果 | 手入力・転記作業を削減 |

**データ入力元（要検討）:**
- 社労士がアプリに手入力
- 企業がアップロードした書類（賃金台帳等）からAIが抽出
- 企業が専用フォームに入力
- 既存の労務管理ソフト（freee、MF等）と連携
