# キャリアアップ助成金 申請支援アプリ

社労士事務所向けのキャリアアップ助成金（正社員化コース）申請支援Webアプリケーション

## 概要

生成AIを活用して社労士事務所の事務負担を最小化し、キャリアアップ助成金の申請をスムーズに完了させることを目指すプロジェクトです。

## 主要機能（予定）

- **図解生成**: 企業向けの制度説明・フロー図を自動生成
- **書類チェック**: 提出書類の不備・漏れをAIが検出
- **書類作成**: フォーマットに沿った申請書類を自動生成
